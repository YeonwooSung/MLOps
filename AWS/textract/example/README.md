## Amazon Textract Code Samples

This directory contains example code snippets showing how Amazon Textract 
and other AWS services can be used to get insights from documents.

All of the codes are from 
[aws github 
repo](https://github.com/aws-samples/amazon-textract-code-samples/tree/master).

Credits @AWS

## Usage

python3 01-detect-text-local.py

For examples that use S3 bucket, upload sample images to an S3 bucket and update variable "s3BucketName" in the example before running it.
